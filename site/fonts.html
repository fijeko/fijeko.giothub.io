Responsive fonts

After googling a lot I found this sample to make font responsiv.

1 mm = 0.03937inch
1 in = 25.4 mm 
	
	html {
		font-size:14.1732pt;       /* = 5mm 0.19685in*/
		line-height:14.1732pt;
		margin : 0;
		padding : 0;
	}
	
	body {
		font-size : 65.2%;         /* 5 * 0.60 = 3mm */                   
		line-height : 1.5em;
	}
	
	p  { font-size: 1.2em; }
	h1 { font-size: 2.0em; margin-top: 1em;}
	h2 { font-size: 1.6em; }
	h3 { font-size: 1.5em; }
	h4 { font-size: 1.3em; }
	h5 { font-size: 1.2em; }
	h6 { font-size: 1.1em; text-align: right;}
	
	/* 959px an up */
	@media only screen and (min-width: 959px) {  
		p  { font-size: 1.2em; }
		h1 { font-size: 2.4em; margin-top: 1em;}
		h2 { font-size: 2.2em; }
		h3 { font-size: 1.9em; }
		h4 { font-size: 1.7em; }
		h5 { font-size: 1.5em; }
		h6 { font-size: 1.3em; }
	}

explanation:

HTML elements inherit font size from parent, and if I give them size in percent or em units the size of font is calculated based on parent size. So if parent has size of 16px and children of 50%. Children font size will be 8px.

Size of pixel vary between devices, but size of point(pt) is 1/72 of inch or cca 0.35 mm on every device.

I start giving my html fonts fixed size of 14pt and that is cca 5mm or 0.2 in
I set body font-size to 60% of that and that is first calculation of font-size. Now my base font size is cca 3mm or 0.12 in.

I'm not sure why this works but works and I have same size of fonts on every device.
I've experimented with different sizes and finally find out that 14pt is good. Bootstrap use this too.

Next I set font size for paragrafs and headers again relative to body size of 3mm.
Also I use media query to set font size larger for wider screen.
